Training and Placement Office Portal

A full-stack web application designed to manage and streamline the placement activities of a college. This system helps students, recruiters, and administrators interact efficiently through a centralized platform.

🚀 Features
👨‍🎓 Student Module
Create and manage profile
Upload resume
View job/internship opportunities
Apply for jobs
Track application status
🏢 Recruiter Module
Post job/internship openings
View student applications
Shortlist candidates
🛠️ Admin / TPO Module
Manage students and recruiters
Approve/reject job postings
Schedule placement drives
Monitor placement statistics
